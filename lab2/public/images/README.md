# Зображення учасників команди

У цій папці потрібно розмістити фотографії учасників команди:

- `kisurin.jpg` - Фото Кісуріна Кирила
- `maksymenko.jpg` - Фото Максименка Романа
- `zub.jpg` - Фото Зуба Максима
- `milishchuk.jpg` - Фото Міліщука Сергія
- `khmeliuk.jpg` - Фото Хмелюк Діани
- `shyrynha.jpg` - Фото Ширинги Андрія

## Вимоги до фотографій:
- Формат: JPG, PNG
- Розмір: бажано квадратні (1:1) для найкращого відображення
- Якість: хороша якість, але не надто великий розмір файлу
- Фон: бажано світлий або нейтральний

Після додавання фотографій сайт автоматично відобразить їх на головній сторінці та на особистих сторінках учасників.
